# 🎶 PYMusicBot V2
A music bot designed to be robust and easy to use

# 📌 Features
- Easy installation
- Uses [yt-dlp](https://github.com/yt-dlp/yt-dlp) and [FFmpeg](https://ffmpeg.org/) (plays almost anything)
- Uses slash commands, so it's easy to use
